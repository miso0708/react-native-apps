# Calculator
## 프로젝트 소개
MAC OS 계산기 클론 코딩 구현


## 중요 기능 순서
* 필요한 state 먼저 세팅 (input, currentOperator, result, tempInput, tempOperator)
* flex에 따라서 레이아웃을 배치하여 계산기 UI를 구현
* 계산 로직 구현
* UI와 로직 코드를 커스텀 훅으로 나눠서 리팩토링 (가독성 증가)
